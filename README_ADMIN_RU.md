# Админ каталог Mini App

Это отдельная админ-панель для управления клиентским проектом **«Демо Каталог Лайт Мини Апп»**.

## Что можно редактировать
- Баннеры на главной (добавление, удаление, порядок)
- Статьи на главной (добавление, удаление, порядок)
- Каталог и подкаталоги (разделы, названия, картинки)
- Карточки товаров (фото, характеристики, тексты, цены, теги)

## Ограничения
- Основные тексты `aboutText`, `productionText`, `paymentText`, `contactsText` открыты только для просмотра (по ТЗ).

## Как использовать
1. Откройте админку по ссылке (или через `/admin` в админ-боте).
2. Внесите правки.
3. Нажмите `Сохранить черновик` (локально в браузере).
4. Скачайте `config.json`, `categories.json`, `products.json`.
5. Загрузите эти файлы в репозиторий клиентского mini app.

## Публикация
Кнопка `Отправить пакет в бота` отправляет в бот сводку изменений. Полные JSON передаются через скачивание файлов.
